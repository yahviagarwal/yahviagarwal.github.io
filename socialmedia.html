Social Media X Ethics

Link to Bluesky Bot

Text-Only Post

How I accomplished the post:

I logged into my Bluesky account through the API and published a simple text message. This demonstrated authentication and the ability to send a plain post.

Design Choice: I kept it minimal with a simple "Hello World" to verify functionality before adding complexity.

Multimedia Post

How I accomplished the post:

I uploaded an image file to Bluesky and attached it to a post with a caption. This required uploading the image first, then embedding it into the post record.

Design Choice: I paired a meme image with a short caption to test how media renders in the feed.

User Interaction (Like)

How I accomplished the interaction:

I liked another user’s post by referencing its unique URI and sending a “like” action through the API instead of manually clicking the button.

Design Choice: Using the URI ensures precision. I chose liking instead of replying to demonstrate interaction while keeping it simple.

External Package Integration

How I accomplished the post:

I used the yfinance library to fetch the current AAPL stock price and dynamically insert it into the post. Instead of hardcoding a value, the content updates automatically.

Design Choice: Stock data changes daily, clearly demonstrating automation and real-world API integration.

Reflection: In-Class Discussion & Pre-Class Readings

One discussion question that shaped my thinking was whether a bot is inherently deceptive, or whether morality depends on its design and disclosure. This shifted my thinking toward transparency and intent. A clearly labeled informational bot feels ethically different from one pretending to be human.

Another discussion focused on whether bots should have equal participation rights as humans. Automating a “like” showed me how easily engagement can scale. While a single action seems harmless, automation at scale can distort public discourse.

The reading Do Social Media Bots Dream of Electric Sheep? helped me see that bots vary widely in function. Informational bots and malicious influence bots may share technical structures but differ ethically based on impact and intent.

Lingering Question

Should platforms require mandatory bot labeling? Transparency seems ethically ideal, but enforcement is difficult, especially with malicious actors.
